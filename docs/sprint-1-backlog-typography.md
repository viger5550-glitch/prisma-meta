# Sprint 1 Backlog: Typography Rollout

## Цель спринта

Внедрить единый типографический стандарт для кнопок и коротких UI-лейблов:
- `font-weight: 500` (`font-medium`);
- без `semibold/bold` в CTA и label-кнопках.

Ограничение: документ плановый, без код-реализации в этом шаге.

## Scope

- согласовать финальный типографический контракт;
- описать точки применения в UI-компонентах;
- зафиксировать критерии приемки и QA-проверки;
- подготовить список задач для следующей реализации.

## Backlog (готово/не готово)

- [ ] Подтвердить источник истины токенов (`ui-tokens-spec.md`).
- [ ] Подтвердить токен `font.weight.ui.label = 500`.
- [ ] Подтвердить правило: кнопки используют только `font-medium`.
- [ ] Подтвердить правило: короткие лейблы в карточках используют `font-medium`.
- [ ] Зафиксировать список компонентов первой волны:
  - `Button`
  - `Card` (кнопка/лейблы)
  - `Header` (короткие nav-label при необходимости)
- [ ] Подготовить checklist визуальной приемки для desktop/mobile.
- [ ] Подготовить checklist accessibility (контраст, читаемость, фокус).
- [ ] Подготовить список экранов для smoke-проверки:
  - мета-страница
  - 1-2 продуктовых лендинга
  - success/cancel сценарии CTA

## Критерии приемки (Definition of Done)

- типографические правила утверждены командой;
- все места применения перечислены без пробелов по coverage;
- есть единый QA-checklist для ручной проверки;
- scope реализации следующего спринта зафиксирован и не содержит лишних задач.

## Out of Scope

- рефакторинг дизайн-системы за пределами типографики;
- изменение цветовой палитры;
- изменение структуры страниц;
- интеграция новых UI-библиотек.
